# CTF

CTFはセキュリティ技術を用いて隠されたフラッグを探し出し得点を競うコンテストのこと

主な種類
・jeopardy形式
	初心者向け、カテゴリ別に問題があり問題を解いていく

・attack&defense形式
	難易度高め、チームごとにサーバーを持ち
                ・自チームのサービスを防御
                ・他チームの脆弱性を攻撃
                を同時に行う

主なカテゴリ
・web
    webアプリケーションの脆弱性を利用してサーバから情報を入手する

・crypto
    暗号化された情報から法則や脆弱性を見つけて隠された情報を復元する

・network
    ネットワーク通信をキャプチャして不審なやり取りの通信などを解析する

・osint
    インターネット上の公開情報から手がかりを収集して情報を特定する

・forensics
    削除ファイルや破損データを調査して証拠となる情報を抽出する

・reversing
    プログラムを解析して動作や処理を逆算して隠された情報を探る

・stegano
    画像、音声、動画などに隠された情報を解析して必要な情報を取り出す

・pwn
    プログラムの脆弱性を悪用して不正権限を取得して情報を奪取する

・misc
    他のジャンルに属さない知識系、パズル系など様々な問題が出題される
